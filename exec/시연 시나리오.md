로그인 → 메인 화면 이동

- 오늘의 현황: 사용자의 카페인, 당, 소비액, 칼로리 현황 제공
- 현재 주문 현황: 가게가 주문 수락 후 실시간 주문 상태 현황 제공
- 음료 추천: 현재 위치와 오늘의 현황 기반 마실 수 있는 음료 목록 추천
